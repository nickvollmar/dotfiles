https://www.sublimetext.com/download

Usage: from this directory, execute the following commands.

```
mkdir -p "$HOME/Library/Application Support/Sublime Text/Packages/User"
ln -s "$PWD/Preferences.sublime-settings" "$HOME/Library/Application Support/Sublime Text/Packages/User/Preferences.sublime-settings"
ln -s "$PWD/Default (OSX).sublime-keymap" "$HOME/Library/Application Support/Sublime Text/Packages/User/Default (OSX).sublime-keymap"
```
